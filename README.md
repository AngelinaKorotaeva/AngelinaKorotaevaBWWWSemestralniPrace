# Cat Café Website

Cat Café Website is a semester web project created with HTML, CSS, and JavaScript.

The website represents a cat café and includes several pages for visitors, such as information about the café, cat adoption, reservations, payment, and sign-in functionality.

## Features

- Multi-page website structure
- Cat café presentation page
- Cat adoption page
- Reservation page
- Payment page
- Sign-in page
- Custom styling with CSS
- Basic JavaScript functionality
- Image assets and visual design

## Technologies

- HTML
- CSS
- JavaScript

## Project Structure

```text
├── css/
├── img/
├── CateCafe.html
├── GetACat.html
├── Pay.html
├── Rezervation.html
├── SignIn.html
└── script.js
```

## How to Run

Open CateCafe.html in a web browser.

No installation is required.

Purpose

The purpose of this project is to demonstrate basic web development skills, including page layout, styling, navigation, and simple interactive functionality.
